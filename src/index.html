2.2.1 Application Programming Interface

  1. Display the feedback form.
    URL
    /feedback?lab_name=:lab_name&exp_name=:exp_name&key=:key OR /feedback?lab_name=:lab_name&key=:key
    Method
    GET
    (no term)
    URL Params Required: lab_name=[string], key=[string] Optional: exp_name=[string]
    (no term)
    Success Response
    • Code: 200
    • Content: { lab_name: 'lab_name', exp_name: 'exp_name', questions: [{name: 'question', type: 'radio | checkbox | textbox', options: 'list of options' | 'size of text'}, ….] }
    (no term)

    Error Response:

    • Code: 401 UNAUTHORIZED
    • Content: { error : "Wrong Key configured" }

    OR

    • Code: 400 Bad Request
    • Content: { error : "lab name is mandatory and experiment is optional" }
  2. Save the feedback form. The authentication for the POST is sent as part of header.
    URL
    /feedback
    Method
    POST
    (no term)
    Data Params { lab_name: 'lab_name', exp_name: 'exp_name', questions: [{name: 'question', answers: [ans1, …] }
    (no term)
    Success Response
    • Code: 200
    • Content: { status: "success" }
    (no term)

    Error Response:

    • Code: 401 UNAUTHORIZED
    • Content: { error : "Wrong Key configured" }

    OR

    • Code: 400 Bad Request
    • Content: { error : "some of the parameters are not part of the request" }
  3. Display the feedback form.
    URL
    /feedback_usage_by_workshop?ip=:ip&date=:date&key=:key
    Method
    GET
    (no term)
    URL Params Required: ip=[string] date=[string], the format of the date string is in 'mm-dd-yyyy' format. key=[string]
    (no term)
    Success Response
    • Code: 200
    • Content: { usage: 'usage_number' }
    (no term)

    Error Response:

    • Code: 401 UNAUTHORIZED
    • Content: { error : "Wrong Key configured" }

    OR

    • Code: 400 Bad Request
    • Content: { error : "IP and Date are mandatory and date should be in 'mm-dd-yyyy' format" }
  4. Display the feedback form.
    URL
    /feedback_dump?date=:date&key=:key
    Method
    GET
    (no term)
    URL Params Required: date=[string], the format of the string is in 'mm-dd-yyyy' format key=[string]
    (no term)
    Success Response
    • Code: 200
    • Content: { feedbacks: [{lab_name: 'lab_name', exp_name: 'exp_name', date: 'mm-dd-yyyy', IP: 'ip', responses: [{question: 'name', answers: ['val1',..] }, …] }, …] }
    (no term)

    Error Response:

    • Code: 401 UNAUTHORIZED
    • Content: { error : "Wrong Key configured" }

    OR

    • Code: 400 Bad Request
    • Content: { error : "Date is mandatory and should be in 'mm-dd-yyyy' format" }
